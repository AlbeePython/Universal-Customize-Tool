## 📖 О проекте

**UCT (Universal Customize Tool)** — это мощный, легковесный и интуитивно понятный инструмент для глубокой персонализации **Windows 11**. 

Многие пользователи столкнулись с тем, что новая ОС от Microsoft навязывает спорные решения: перегруженное контекстное меню, навязчивый поиск Bing и встроенный ИИ Copilot. **UCT возвращает контроль в руки пользователя**, позволяя отключать лишнее и возвращать привычный функционал в один клик.

> [!IMPORTANT]
> **Безопасность прежде всего:** Все изменения вносятся через официальные ключи реестра (WinReg API) и системные политики. Программа не патчит системные файлы (.dll/.exe), что гарантирует стабильность системы после обновлений.

## ✨ Основные возможности (Release T)

В текущей версии реализована сетка модификаций (**Grid 2x3**), охватывающая самые востребованные запросы:


| Модуль | Описание | Метод реализации |
| :--- | :--- | :--- |
| **Context 10 Styler** | Возвращает классическое меню правой кнопки мыши (как в Win 10). | CLSID Registry Hack |
| **No Bing Search** | Отключает облачные результаты и рекламу в поиске меню "Пуск". | Explorer Policies |
| **Bye Bye Copilot** | Полностью удаляет иконку и функции ИИ Copilot из системы. | WindowsCopilot Policy |
| **Show Seconds** | Включает отображение секунд в системных часах панели задач. | Advanced Explorer Key |
| **No Shortcut Arrow** | Эстетическое удаление стрелочек у всех ярлыков на рабочем столе. | Shell Icons Overlay |
| **MiniBinKT** | Компактная корзина прямо в панели задач (внешний модуль). | External Integration |

## 🛠 Техническая архитектура
Проект построен на **PyQt6**:
* **UI/UX:** Использование кривых Безье `OutQuint` для анимаций.
* **Graphics:** Эффект `QGraphicsBlurEffect` для динамического размытия контента.
* **Registry Engine:** Безопасная работа с `HKEY_CURRENT_USER` и `HKEY_LOCAL_MACHINE`.
